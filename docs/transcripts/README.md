# Video transcripts

Raw transcripts of the openDog V3 build videos. These are the **source material** I'll turn into `docs/ASSEMBLY.md`.

## How to add one (no tools to install)
1. Open the video on YouTube.
2. Click **"...more"** to expand the description.
3. Scroll down to **"Show transcript"** and click it.
4. In the transcript panel, click the **⋮** menu → **"Toggle timestamps"** *off* (cleaner text).
5. Select all the transcript text → copy.
6. Paste into the matching `.txt` file below (create it if needed), or just paste into chat and I'll file it.

Filename convention: `NN-short-slug.txt` (matches the numbers below).

Captured transcripts live in `raw/` as `NN.md`. They are timestamped (and include sponsor ad-reads / `[Music]` markers — ignore those). All six are processed into `../ASSEMBLY.md`.
